# VKarmani Desktop

Tauri + Rust + React/Vite desktop client for VKarmani.

## Быстрый старт

```
powershell
npm install
npm run tauri dev
```

## Запуск в dev с TUN

Для TUN в режиме разработки запускай PowerShell **от имени администратора**:

```powershell
cd "D:\GIT\VKarmani"
.\START_VKarmani.bat
```
